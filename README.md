# x-concole

Color console

## Install

```sh
$ npm install x-console
```

x-concole depends on [Node.js](http://nodejs.org/).

## Usage

```js

var console = require('x-console');

console.log('default');
console.info('blue');
console.warn('yellow');
console.error('red');

```
